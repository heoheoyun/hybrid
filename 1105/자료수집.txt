1. Create a comprehensive financial investment platform called "NVIDIA Asset" - a web application for Korean investors interested in NVIDIA stock, products, and technology.

=== PROJECT OVERVIEW ===

Target Users: Korean NVIDIA investors and technology enthusiasts
Primary Language: Korean UI with English technical terms (e.g., "GPU", "P/E Ratio", "AI", "GeForce RTX")
Purpose: Real-time stock information, product showcase, company news, and investor resources

=== DESIGN SYSTEM ===

Color Palette:
- Primary: NVIDIA Green #76B900 (buttons, accents, positive changes)
- Secondary: Black #000000 (text, headers)
- Background: Dark Gray #1A1A1A (main background)
- Surface: White #FFFFFF with 10% opacity (cards, content areas)
- Accent Red: #FF4444 (negative price changes)
- Gradient: Linear gradient from #76B900 to #000000 for hero sections

Visual Style:
- Dark mode by default with technology-focused aesthetic
- Glassmorphism effects (backdrop-blur, subtle borders)
- Modern sans-serif typography (use Inter or similar)
- Smooth animations and transitions (300ms ease-in-out)
- Responsive design: Mobile (320px+), Tablet (768px+), Desktop (1024px+)

=== TECHNOLOGY STACK ===

Frontend: React 18 with TypeScript
UI Framework: shadcn/ui components with Tailwind CSS
State Management: React Query (TanStack Query) for API data caching
Charts: Recharts for stock visualizations
Icons: Lucide React
Animations: Framer Motion for scroll animations
HTTP Client: Axios with interceptors
WebSocket: Socket.io-client for real-time updates
